# Library-Management-System

A simple web-based Library Management System built with HTML and JavaScript. This project allows users to manage a collection of books in a library, providing essential features such as adding, searching, and removing books, as well as tracking their availability.

---

## Project Description

The Library Management System is designed to help users efficiently handle library book inventories through an interactive web interface. It is suitable for small libraries or as a learning tool for web development. The system is lightweight, requiring no backend or database setup, and all operations are performed on the client side.

---

## Features

- **Add New Books:** Input book details and add them to the collection.
- **Search Books:** Find books by title, author, or other criteria.
- **Remove Books:** Delete books from the collection.
- **Track Availability:** Mark books as available or checked out.
- **User-Friendly Interface:** Intuitive controls for managing the library.

---

## Setup Instructions

1. **Clone the Repository:**
   ```sh
   git clone https://github.com/Subarnak18/Library-Management-System.git
   cd Library-Management-System
   ```

2. **Open the Application:**
   - Locate the main HTML file (typically `index.html`).
   - Double-click or open the file in your preferred web browser.

3. **No additional installation or backend setup is required.**
   - All data is managed in-browser and will reset upon refresh unless you add browser storage.
